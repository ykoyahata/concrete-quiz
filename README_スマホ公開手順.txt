コンクリート診断士 過去問AI（PWA版）

【スマホだけで公開する最短手順：Netlify Drop】
1. このZIPをスマホにダウンロード
2. ファイル管理アプリでZIPを展開
3. Chromeで https://app.netlify.com/drop を開く
4. Netlifyに無料登録またはログイン
5. 「Browse to upload」等から、展開したフォルダ内の index.html / manifest.webmanifest / service-worker.js / icons フォルダをまとめてアップロード
6. 発行された https://～.netlify.app のURLをChromeで開く
7. Chrome右上の︙から「アプリをインストール」または「ホーム画面に追加」を選ぶ
8. 一度オンラインで開いた後は、基本的にオフラインで利用可能

※ Google Driveは保管場所には使えますが、PWA公開先には向きません。
※ 学習記録は端末・ブラウザ内に保存されます。
