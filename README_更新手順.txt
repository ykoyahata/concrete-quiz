【学習履歴について】
・従来の保存キーは変更していないため、GitHub上のHTMLを置き換えても、同じ端末・同じブラウザでは回答済み、正答率、間違い、後で復習の記録は原則そのまま残ります。
・念のため、今後は「苦手分析」→「バックアップを保存」からJSONファイルを保存できます。
・バックアップから復元すると、現在の学習記録はバックアップの内容で置き換わります。

【更新手順】
1. GitHubの concrete-quiz リポジトリを開く
2. ZIP内の6ファイルをリポジトリ直下へアップロードする
3. 同名ファイルの置き換えを確認し、Commit changes を押す
4. GitHub Pagesへの反映を1〜10分ほど待つ
5. ブラウザまたはホーム画面のアプリを完全に閉じ、もう一度開く
6. 古い画面のままなら、ブラウザでページを再読み込みしてからアプリを開き直す

※ホーム画面アプリを削除しなくても、今回のHTML更新は反映できます。
